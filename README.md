# Rental Budget Generator - R.M. Real Estate

A simple, automated system for generating residential rental budgets (houses, apartments, and studios), developed to meet the operational and commercial needs of **R.M.** Real Estate.

## Objective

To automate and standardize the creation of monthly rental budgets, reducing manual errors, streamlining customer service, and facilitating the professional presentation of proposals.

## Main Features

- Quick registration of customer data (name, contact, CPF)
- Selection of property type: House, Apartment, or Studio
- Automatic calculation of base rent with specific rules per property type
- Inclusion of additional costs (condominium fees, property tax, administrative fees)
- Application of customized discounts or surcharges
- Detailed budget display with breakdown of values
- Basic export of the budget to a text file (prepared for future PDF generation)

## Technologies Used

- **Language**: Python 3
- **Paradigm**: Object-Oriented Programming (inheritance and polymorphism)
- **Libraries**: datetime (native), os (native)
